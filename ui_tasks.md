# UI Tasks: MVP Система Бронирования

## 📌 Контекст проекта (Для AI ассистента)
Проект представляет собой MVP системы бронирования услуг. В качестве основного стека используется **Next.js (App Router)**, **Tailwind CSS** и **shadcn/ui**. Бэкендом (базой данных и аутентификацией) выступает **PocketBase** (деплоится на Coolify), а платежной системой — **Stripe**. 

**Текущая задача:** Сверстать все необходимые страницы и UI-компоненты с использованием **моковых (фейковых) данных**, чтобы подготовить фронтенд к последующей интеграции с PocketBase и Stripe.
**Подход к разработке:** Необходимо строго следовать методологии **Atomic Design**. Структура папки `components` должна быть разделена на `atoms`, `molecules`, `organisms`, `templates`.

Используй команду `npx shadcn-ui@latest add <component>` для установки нужных базовых компонентов (в папку по умолчанию или в `atoms`). Разрабатывай страницы так, чтобы они выглядели "дорого", современно (с hover эффектами, плавными анимациями, красивой типографикой).
